# Project 3 Starter — Community Programs Hub

This is a starter codebase for **ARTI-324A Project 3: Reconstruction & Intervention**.

## What this includes
- Semantic-ish structure (intentionally imperfect)
- Repeated card pattern
- Expand/Collapse panels
- Flip "back view"
- Menu toggle
- FAQ toggle
- Form feedback (simple)
- Optional filters (present, minimal)

## notes
This repo is designed to be:
- understandable with inspection
- slightly messy
- full of small improvement opportunities

Do NOT add frameworks for the course version.
